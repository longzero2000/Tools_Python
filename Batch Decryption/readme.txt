Giải mã hàng loạt md5, sha356, ...
1. Đầu tiên cần mạng để sử dụng chương trình
2. Cần lưu trữ các mã theo mẫu file "data.txt"
3. Mở đầu phần mềm cần nhập mật khẩu
4. Tiếp theo đó là nhập loại mã hóa để giải mã (chữ thường)
5. Chờ quá trình giải mã hoàn tất và xuất file đã giải mã
--------------------------------------------------------
Batch decode md5, sha356, ...
1. First need the network to use the program
2. Need to store the code according to the file template "data.txt"
3. Start the software to enter the password
4. Next is to enter the encryption type to decrypt (lowercase)
5. Wait for the decryption process to complete and export the decrypted file
